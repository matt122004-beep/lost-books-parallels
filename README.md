# Lost Books of the Bible — Parallels Web App

A browser-based reference tool showing textual parallels between the Apocrypha/Pseudepigrapha and canonical Bible texts.

## Books Covered
- 1 Enoch, 2 Enoch
- Book of Jubilees
- Gospel of Thomas
- Gospel of Peter
- Wisdom of Solomon
- Sirach (Ecclesiasticus)
- 1 & 2 Maccabees
- Tobit, Judith
- And more

## Features
- Side-by-side parallel text view
- Filter by book, theme, or keyword
- Dark scholarly theme

## Usage
Open `index.html` in any modern browser — no server required.

## Deployment
Static file — deploy to GitHub Pages, Netlify, or any web host.
